# Re-vents

Event manager for users

# Techs

- React
- Redux
- Firebase

# Run

- Fill in `env.example` with corresponding signature
- **With Docker**

  - Run `docker build . -t example-tag`
  - Run `docker run -d -p 3000:3000 example-tag`
  - Open browser on http://localhost:3000

- **Without Docker**
  - Run `yarn install` or `npm install`
  - Run `yarn start` or `npm run start`
